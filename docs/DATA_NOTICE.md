# Data Notice

This public repository must not contain confidential, proprietary, customer, or employer-provided data.

Allowed:

- Synthetic fixtures generated by `generate_dummy_data.py`.
- Synthetic non-routine maintenance prediction artifacts generated by `generate_mock_nr_artifact.py`.
- Broad schema and scale matching, such as row counts, column names, and approximate numeric ranges.
- Publicly available airport metadata only when its license permits redistribution.
- Small hand-written examples created specifically for this public project.

Not allowed:

- Private airline extracts.
- Derived real-input bundles.
- Row-level transformations of private operational data.
- Real aircraft registrations, task identifiers, maintenance panels, or route patterns from private sources.
- Historical operational schedules or maintenance logs from a private source.
- Generated simulation outputs from private scenarios.
- Pickled objects, trained model weights, or dashboards produced from private data.
- Git history copied from a private repository.

Before publishing or pushing, run:

```powershell
rg -n -S "KLM|Air France|real_777|OneDrive|private|confidential|proprietary|PH-B|FlightLegs"
git status --short
```

If either command shows private-source artifacts, remove them before publishing.
