# CV Positioning

This public repository is ready to support job applications as a synthetic-data portfolio project.

## Safe Project Summary

Built a public aircraft maintenance simulation and ML evaluation platform using synthetic operational data. The project includes reproducible experiment tracking, policy comparison dashboards, FastAPI endpoints, Docker packaging, CI checks, grounded KPI reporting, local artifact retrieval, and runtime monitoring.

## CV Bullets

- Built a synthetic aircraft maintenance simulation evaluation platform with reproducible experiment records, fixed-seed policy comparisons, and local/optional MLflow-compatible tracking artifacts.
- Developed a Dash dashboard and FastAPI service for comparing maintenance policy rungs across KPI proxies including delay, interval spillage, completion factor, and uncovered non-routine labour.
- Packaged the public workflow with Docker and GitHub Actions CI covering synthetic fixture generation, tracked experiment smoke tests, API health checks, unit tests, and private-data safety scanning.
- Added grounded analyst reports and local retrieval over generated KPI/profile/report artifacts, with runtime API metrics for request count, failures, search usage, and latency.

## Interview Talking Points

- Data boundary: the public repo intentionally uses shape-matched synthetic fixtures and excludes private operational data, private Git history, and derived real-input bundles.
- Reproducibility: the default experiment is deterministic by seed and writes explicit run metadata, metrics, and summaries.
- MLOps signal: the repo demonstrates experiment tracking, artifact management, CI, API packaging, dashboarding, and monitoring in a compact public workflow.
- Product/stakeholder signal: the dashboard and grounded report convert technical KPIs into reviewable policy-comparison evidence.

## Claims To Avoid

- Do not claim live production deployment.
- Do not claim the public repo contains or evaluates real airline data.
- Do not claim Kubernetes, high-traffic service ownership, or production incident response.
- Do not present synthetic KPI values as operational results.

## Strong Portfolio Sentence

I built a public synthetic-data aircraft maintenance ML evaluation platform with reproducible experiment tracking, policy comparison dashboards, FastAPI service endpoints, Docker/CI packaging, grounded KPI reporting, local experiment retrieval, and runtime monitoring.
